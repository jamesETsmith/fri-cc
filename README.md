## Installation

### Requirements
- C++20 Compiler
- cmake
- BLAS and LAPACK
- Eigen (packaged with project)
- pybind11 (packages with project)
- Python >= 3.6
    - PySCF
    - pytest

### From source

```bash
mkdir build && cd build
CXX=<your desired c++ compiler> cmake ..
```
